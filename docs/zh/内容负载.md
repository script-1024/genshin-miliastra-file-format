## GIA - 资产文件
```c#
public partial class GiaFile : GiFile {
    public override GiFileType Type => GiFileType.Gia;

    // id = 1, type = LENGTH
    public IList<Asset>? Assets;
    
    // id = 2, type = LENGTH
    public IList<Asset>? DependentAssets;
    
    // id = 3, type = LENGTH
    public string? ExportInfo;
}
```

### Assets
- 类型：列表
- 一个或多个资产定义对象
- 通常是玩家在「资产导入导出界面」中显式导出的元件

### DependentAssets
- 类型：列表
- 依赖的资产对象，此项可能不存在
- 通常是那些在导出时没被玩家选中，但其他组件必须依赖它才能运作的元件

### ExportInfo
- 类型：字符串
- 导出信息（可能用于月之三版本即将实装的资产分享平台中显示作者名称？）
- 游戏似乎不会校验此字段的内容，即便乱填或移除也不影响加载
- 默认格式是 `{玩家UID}-{导出时间戳}-{文件GUID}-\{导出时名称}.gia`
