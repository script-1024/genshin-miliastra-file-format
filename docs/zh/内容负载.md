# 内容负载
下方链接主要介绍所有文件类型（`.gip`, `.gil`, `.gia`, `.gir`）的根标签结构，至于各别子标签的字段格式将在对应文档中详细说明。

*字段名称仅供参考。因为游戏源代码经过混淆，我们无法得知它们的真实名称和用途！*

## GIP - 项目文件
> 此段落仍在编辑和测试中

<br/>

## GIL - 关卡文件
在「我的奇域」中创建的关卡，其中包含各式各样的关卡数据，给本项目的工作进展带来严重阻碍 💀

请见对应介绍页面：🔗[链接](/docs/zh/内容负载/GIL.md)

<br/>

## GIA - 资产文件
透过「资产导入导出」功能生成的文件，通常包含一个或多个由玩家勾选导出的资产定义

请见对应介绍页面：🔗[链接](/docs/zh/内容负载/GIA.md)

<br/>
